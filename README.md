# Автоматизация обмена библиографическими данными

Скрипт на Python для автоматизации выгрузки и загрузки данным между Автоматизированной библиотечно-информационной системой (АБИС) и Сводным электронным каталогом (СЭК) библиотечных систем.

Основной функционал:
1. **Безопасная распаковка данных**: извлечение конфигураций и параметров из зашифрованных 7-z архивов.
2. **Парсинг и генерация сценариев**: автоматическое чтение конфигурационных файлов и динамическое формирование исполняемых '.bat' сценариев для консольных утилит АБИС.
3. **Очистка конфиденциальной информации**: автоматический поиск и удаление авторизационных логов и технических метаданных из лог-файлов после завершения сессии.
4. **Оповещение по email**: настройка прокси соединения и отправка отчетов об успешных операциях на электронную почту с помощью SMTP-протокола.
